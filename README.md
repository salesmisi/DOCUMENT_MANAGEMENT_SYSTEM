# DOCUMENT_MANAGEMENT_SYSTEM

This project is a web-based Document Management System (DMS) for managing files, folders, and user activities.

## 🚀 Getting Started

1. Install dependencies:
   npm install

2. Run the development server:
   npm run dev

## 📌 Features
- File upload and management
- Folder organization
- Activity logs
- User authentication
